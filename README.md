# Node 

Esta aplicación genera archivos .txt o impresiones en consola de tablas de multiplicación

Ejecutar el comando
``
npm install
``

## Para ejecutar el programa se utiliza el siguiente comando
``
 node app [crear, listar] --base (required) --limite(opcional)
 ``

 ## Para más información ejecute el siguiente comando
 ``
 node app --help
 ``
